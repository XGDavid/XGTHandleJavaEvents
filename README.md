# XGTHandleJavaEvents

XGTHandleJavaEvents is a plugin designed to handle events for the XGTJavaBridge, enabling seamless interaction between
Minecraft: Java Edition clients and PocketMine-MP servers.

## Features

- Handles events triggered by Java Edition clients connected via XGTJavaBridge.
- Provides a foundation for extending server functionality and compatibility.

## Requirements

- **PocketMine-MP API**: 5.0.0 or higher
- **Dependency**: [XGTJavaBridge](https://discord.gg/nPDC7FPbw8) (Allows the connection of Minecraft: Java
  Edition clients to PocketMine-MP servers)
